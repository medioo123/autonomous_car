Explication des noms des modèles
________________________________________________________________________________
Tous les modèles ont un nom de la forme suivante :
"TransferNomDuModèle_NombreEpochs_TypeOutput_MéthodeImportation"

NombreEpochs --> précise le nombre d'epochs pour l'entrainement du modèle

Pour les autres caractéristiques, elles sont identiques à celles précisées dans
le "ReadMe.txt" expliquant le nom des Notebooks qui créent ces modèles
(Autonomous_car/TransfertNotebooks/ReadMe.txt)
